# MovDick
vim + docker

## Getting started

### Prerequisite

docker

### Install

```console
git clone https://github.com/myoungone/MovDick.git
make install
```
Logout and re-login to use the installed script.

### Uninstall

```console
make clean
```
Logout and re-login to use the original vim.

### Usage

Just use as regular `vim` or `vi`
